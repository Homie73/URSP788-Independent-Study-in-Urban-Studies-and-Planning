# When the School Bell Rings: Interactive Appendix

This repository hosts the interactive appendix for:

**When the School Bell Rings: Analysis of Non-Motorist Crash Patterns Around Schools in Six Maryland Jurisdictions During School Arrival and Dismissal Periods**

Authors: Homayoon Aghili and Caleb Fan Affiliation: University of Maryland, College Park Course: **URSP788** Independent Study in Urban Studies and Planning Term: Spring **2026**

## Interactive Appendix

The published GitHub Pages appendix includes supplemental comparative maps, interactive charts, and source tables for the six-jurisdiction Maryland non-motorist crash analysis. The appendix is intended to complement the written report and should be interpreted as descriptive and bivariate association evidence rather than causal or exposure-adjusted risk evidence.
