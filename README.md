## Description

The published GitHub Pages appendix includes supplemental comparative maps, interactive charts, and source tables for the six-jurisdiction Maryland non-motorist crash analysis. The appendix is intended to complement the written report and should be interpreted as descriptive and bivariate association evidence rather than causal or exposure-adjusted risk evidence.
